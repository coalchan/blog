nrm——快速切换NPM源

发表于 2014-12-16 更新于 2019-07-20 分类于 技术
nrm是一个 NPM 源管理器，允许你快速地在如下 NPM 源间切换：

安装

npm install -g nrm

使用

列出可选的源

nrm use taobao

Registry has been set to: http://registry.npm.taobao.org/

nrm test npm

npm —- 1328ms

nrm test

npm —- 891ms
cnpm — 1213ms

  • taobao - 460ms
    eu —– 3859ms
    au —– 1073ms
    sl —– 4150ms
    nj —– 8008ms
